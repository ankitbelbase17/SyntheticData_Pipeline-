
# Synthetic Data Pipeline

## Overview
Automates synthetic data generation and validation using vision-language and language models.

## Main Modules
- `robust_scraper.py`: Scrapes and collects data
- `keyword_sampler.py`: Samples relevant keywords
- `mllm_to_vlm_prompt.py`: Converts MLLM output to VLM prompts
- `pipeline_orchestrator.py`: Orchestrates the pipeline workflow
- `model.py`, `edit_model_pipeline.py`, `qwen_vl_processor.py`: Model and processing logic

## Usage
Bash scripts for running each module are in the `bash_scripts` directory.
